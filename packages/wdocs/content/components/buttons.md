# Buttons

Documentation for button components.

Buttons are a fundamental part of any user interface. Here are some examples of the button styles available.

## Primary Button

This is the main call-to-action button.

```vue
<template>
	<button class="px-4 py-2 font-semibold text-white bg-blue-500 rounded-md hover:bg-blue-600">
		Primary Button
	</button>
</template>
```

## Secondary Button

Use this for actions that are important but not the primary focus.

```vue
<template>
	<button class="px-4 py-2 font-semibold text-gray-700 bg-gray-200 rounded-md hover:bg-gray-300">
		Secondary Button
	</button>
</template>
```

## Danger Button

This button indicates a destructive action.

```vue
<template>
	<button class="px-4 py-2 font-semibold text-white bg-red-500 rounded-md hover:bg-red-600">
		Danger Button
	</button>
</template>
```

Welcome to WDocs! This documentation provides everything you need to get started with building your own beautiful and functional documentation sites.

## What is WDocs?

WDocs is a documentation system built on top of Nuxt, offering a VitePress-like experience with the flexibility of the Nuxt ecosystem. It's designed to be simple to use, yet powerful enough to handle complex documentation needs.

### Key Features

- **Vue 3 & Nuxt 3:** Built on the latest technologies for a modern development experience.
- **File-based Routing:** Create pages and routes simply by adding Markdown files.
- **Markdown Extensions:** Supercharge your Markdown with components, syntax highlighting, and more.
- **Extensible:** Easily customize and extend the system to fit your needs.

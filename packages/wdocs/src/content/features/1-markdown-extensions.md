# Markdown Extensions

This documentation system supports several Markdown extensions to enhance content creation.

## Custom Containers

You can use custom containers to highlight information.

:::info
This is an informational message.
:::

:::warning
This is a warning message.
:::

:::danger
This is a danger message.
:::

## Syntax Highlighting

Code blocks are automatically highlighted.

```vue
<template>
  <div>{{ message }}</div>
</template>

<script setup>
const message = 'Hello, Vue!'
</script>
```
